Java

Processing XML file

One java program to handle the KGML formate from KEGG to plain txt.

Example:

For p53 signaling pathway - Homo sapiens (human) [link]

Download the kgml file and the web source code from above link and save as: hsa04115.xml and p53_show_pathway.htm and then run:

java -jar processingXML.jar hsa04115.xml p53_show_pathway.htm output.txt

Here the output.txt is the output.

Download source code , hsa04115.xml , p53_show_pathway.htm , processingXML.jar and output.txt .

Dijkstra Algorithm
One Java program (usage: java -jar problem083.jar matrix.txt) uses Dijkstra's algorithm to solve Euler Project Problem 83

The detailed src files:

lement.java : class uses to define the element of the matrix.

DijkstraAlgorithmEngine.java is the algorithm implementation.

Problem083.java : the main method.
